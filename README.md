# MEG
MEG is messenger grade application 

THis is a custom build application which focuses on D2D ( device to device ) connection which will be anamanonymous ( sender anonymous ) . It can only be   traced back with specific tools.

- The messages will be sent in a pop-up notifiation message like an alert and will dissaper within 10sec - 60 seconds time frame .

- The history in the database will be automatically deleted, only the admin will have access to the database history where there is a log of texts which is purged every 3 months.

- It creates an end-to-end encrytion and makes the data packets neraly impossible to trace or sniff/steal.

- It does not use any interent it works on carrier network and only need internet when installed and customized.
